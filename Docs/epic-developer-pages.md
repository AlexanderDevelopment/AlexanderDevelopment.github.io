# Epic Developer Pages

Epic Developer Portal may request a public application website and a public privacy policy URL for application review or product setup.

Use these production URLs:

- Application website: `https://zweibier-indie.de/dreadway.html`
- Privacy policy: `https://zweibier-indie.de/epic-privacy-policy.html`

## Application Website

`dreadway.html` is a short public page for Epic review. It identifies ZweiBier Indie, lists the intended application purpose, links to current games, and points to the Epic-specific privacy policy.

## Epic Privacy Policy

`epic-privacy-policy.html` documents data that may be processed when a ZweiBier Indie game or application uses Epic account sign-in, Epic Online Services, Epic Games Store features, multiplayer sessions, invites, friends features, achievements, or similar Epic-provided functionality.

The policy must include a plain, clickable email address for privacy questions and data protection inquiries so Epic Games Store review can identify a valid contact method.

Update this page if a game adds or removes Epic features, changes data categories, changes service providers, or starts processing additional Epic-related user data.
